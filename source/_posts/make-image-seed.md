# 教程：如何制作图种(伪装压缩文件为图片)

## 什么是图种？
图种是一种将压缩文件（如ZIP/RAR）隐藏在图片文件中的技术。生成的复合文件既可以被正常显示为图片，又可以作为压缩包提取出隐藏内容。

## 准备工作
1. 一张图片（建议使用JPEG或PNG格式）
2. 一个压缩文件（ZIP或RAR格式）
3. Windows系统可以使用命令提示符，Mac/Linux可以使用终端

## 方法一：使用copy命令（Windows）

1. 将图片和压缩文件放在同一文件夹
2. 打开命令提示符（cmd）
3. 导航到文件所在目录：
   ```
   cd 文件夹路径
   ```
4. 执行合并命令：
   ```
   copy /b 原图片.jpg + 压缩文件.zip 新文件.jpg
   ```
5. 生成的新文件就是图种

## 方法二：使用cat命令（Mac/Linux）

1. 打开终端
2. 导航到文件所在目录
3. 执行合并命令：
   ```
   cat 原图片.jpg 压缩文件.zip > 新文件.jpg
   ```

## 提取隐藏内容

1. 修改图种文件扩展名为.zip或.rar
2. 使用解压软件打开
3. 或者直接使用压缩软件打开图种文件（无需改名）

## 注意事项

1. 原图片文件要放在压缩文件前面
2. 这种方法不适用于所有图片格式，JPEG成功率最高
3. 某些网站可能会重新压缩上传的图片，破坏隐藏数据
4. 请勿用于非法用途

## 验证图种

1. 用图片查看器打开 - 应能正常显示
2. 用压缩软件打开 - 应能看到隐藏文件
3. 用文本编辑器打开 - 文件末尾应能看到压缩文件头标识（如PK for ZIP）
